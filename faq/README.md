# Frequently Asked Questions

## General

**Is my financial data safe?**
Yes. All data is stored locally on your computer. Nothing is sent
to any server except encrypted backups to your own private GitHub
repository (optional, requires your setup).

**Does FinAdvisor sell my data?**
No. FinAdvisor is a local application. We have no server that
receives your financial information.

**What happens if I lose my computer?**
If you set up GitHub backup, you can restore from your encrypted
backup file. Without backup, data would need to be re-entered.
We strongly recommend setting up backup.

## API Keys

**What if I don't set up any API keys?**
FinAdvisor works without any API keys. You will enter financial
data manually and market data will use cached prices.

**Can I add API keys after the initial setup?**
Yes. Go to Settings in your FinAdvisor dashboard at any time
to add, update, or remove API keys.

**Is there a cost to use the API keys?**
Finnhub is free forever. Anthropic has a small per-use cost —
review anthropic.com/pricing and set a spending limit before use.
Coinbase, Kraken, and Gemini APIs are free.

## Technical

**Which operating systems are supported?**
Currently Windows 10 and Windows 11. Mac and Linux support
is planned for a future release.

**What Python version is required?**
Python 3.9 or higher. The installer handles this automatically.
