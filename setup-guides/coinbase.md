# Coinbase API Setup Guide
*Last verified: June 2026*

## Steps

1. Log into your Coinbase account at **coinbase.com**
2. Go to **Settings → API** (coinbase.com/settings/api)
3. Click **Create API Key**
4. Select **Advanced Trade**
5. Choose your portfolio (Primary)
6. Check **View** permission only — no other permissions needed
7. Set IP whitelist to 0.0.0.0/0 (allows all IPs)
8. Click **Create & Download**
9. Save the downloaded JSON file immediately — the private key
   cannot be retrieved again after closing this screen

## Important Note

New Coinbase API keys require up to **48 hours to activate**.
During this time your crypto balance will show as unavailable.
This is normal — no action required.

Free.
